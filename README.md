# Rodar o MySQL usando o Docker Compose para o backend

## Requisitos

- [Docker](https://www.docker.com/get-started) instalado.
- [Docker Compose](https://docs.docker.com/compose/install/) instalado.

## Como rodar

0. **Navegar até o diretório `/demo`:**

   Acesse a pasta onde está o arquivo `docker-compose.yml`.

   ```bash
   cd /demo

1. **Rodar o docker-compose.yml**

   ```bash
   docker compose up -d
